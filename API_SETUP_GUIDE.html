https://www.notion.so/my-integrations

Name: PK Command Center

"Submit" 클릭

2 API 토큰 복사

생성 후 나오는 "Internal Integration Secret"을 복사합니다 (ntn_으로 시작)

3 프로젝트 보드에 연결

Notion에서 "👨🏻‍💻 프로젝트 보드(업무)" 페이지 열기 → 우측 상단 "..." → "연결 추가" → "PK Command Center" 선택

아래 2개를 추가:

NOTION_API_TOKEN = ntn_여기에_복사한_토큰 NOTION_DATABASE_ID = 31a7a0d68bfb8044917ce8cbfd7821e2

2️⃣ Google Calendar API 연동 (10분)

1 Google Cloud 프로젝트 생성 https://console.cloud.google.com/projectcreate

프로젝트 이름: PK Command Center

2 Calendar API 활성화 https://console.cloud.google.com/apis/library/calendar-json.googleapis.com

"Enable" 클릭

3 OAuth 동의 화면 설정 https://console.cloud.google.com/apis/credentials/consent

User Type: "External" → 앱 이름: "PK Command Center" → 이메일 입력 → 저장

4 OAuth 클라이언트 ID 생성 https://console.cloud.google.com/apis/credentials

"+ CREATE CREDENTIALS" → "OAuth client ID" → 앱 유형: "Web application"
Authorized redirect URIs에 추가: https://developers.google.com/oauthplayground

생성 후 Client ID와 Client Secret을 복사

5 Refresh Token 발급 https://developers.google.com/oauthplayground

우측 상단 ⚙️ → "Use your own OAuth credentials" 체크 → Client ID/Secret 입력

Step 1: "Google Calendar API v3" → "calendar.events.readonly" 선택 → "Authorize APIs"

eijconstruction0316@gmail.com 계정으로 로그인

Step 2: "Exchange authorization code for tokens" 클릭 → Refresh token 복사

6 Vercel에 환경변수 등록 https://vercel.com/pentahexagon/peter-command-center/settings/environment-variables

아래 4개를 추가:

GOOGLE_CLIENT_ID = 여기에_클라이언트_ID GOOGLE_CLIENT_SECRET = 여기에_클라이언트_시크릿 GOOGLE_REFRESH_TOKEN = 여기에_리프레시_토큰 GOOGLE_CALENDAR_ID = eijconstruction0316@gmail.com
✅ Google Calendar 연동 완료! Vercel에서 "Redeploy" 하면 실제 일정이 자동으로 표시됩니다.
